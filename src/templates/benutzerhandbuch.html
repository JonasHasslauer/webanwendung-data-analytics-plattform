Benutzerhandbuch

Damit Sie sich auf unserer Webseite einfach zurechtfinden, finden Sie hier ein ausführliches Benutzerhandbuch.
In der Navigationsbar können Sie Ihre Userdaten und das Impressum finden. Hier können Sie auch zwischen den verschiedenen Seiten wechseln.
Wir wünschen Ihnen viel Spaß bei der Benutzung unserer Webseite!
Wenn Sie ein Input-Feld auf der Übersichtsseite anwählen, wird Ihnen dieses nochmal kurz erklärt.

Übersichtsseite

Auf der Übersichtsseite befindet sich der Dateiupload, das Dateiarchiv und die Selektion der Daten.
Für den Dateiupload wählen Sie eine csv-Datei aus, und wählen den Seperator: ist Ihre csv-Datei mit einem ',' oder einem ';' getrennt? Nachdem sie auf klicken, wird die Datei in der Datenbank gespeichert. Sie bekommen direkt Feedback, ob der Upload geklappt hat. Der Upload klappt nicht, wenn es sich bei der Datei nicht um eine csv-Datei handelt, oder wenn schon eine Datei mit demselben Namen existiert. Um die hochgeladene Datei nun im Dateiarchiv sehen zu können, klicken Sie auf
Im Dateiarchiv finden Sie alle Dateien, die Ihrem Benutzernamen zugeordnet sind. Sie haben mit auch die Möglichkeit, die ausgewählte Datei aus der Datenbank zu löschen.
Wählen Sie eine Datei aus (im Beispiel: Sacramentoappartment), wird diese rechts unter Selektion der Daten angezeigt. Nun können Sie die Datei anschauen und selektieren. Dies wird im nächsten Abschnitt noch genauer erläuert.

Direkt unter der angezeigten Tabelle finden Sie die Möglichkeit, Ihre Daten zu selektieren.
Spaltenfilter:
In der ersten Zeile haben Sie die Möglichkeit, einzelne Spalten zu selektieren. In dieser Beispieldatei wäre eine mögliche Eingabe: 'street,city,beds,baths,type', um sich nur noch diese Spalten anzeigen zu lassen. Die einzelnen Werte müssen mit einem ',' getrennt und ohne Leerzeichen eingegeben werden.
Wertefilter:
Wenn Sie nach bestimmten Werten filtern möchten, haben Sie dazu im 2. Block die Möglichkeit: Sie können den Spaltennamen, den Operator und den gewünschten Wert wählen. Im Beispiel könnte man beispielsweise 'beds>2' filtern, und würde nur noch die Zeilen sehen, in denen der Wert von beds größer als 2 ist. Sie können bei Operator auch statistische Werte wählen: Median, unteres Quartil und oberes Quartil. Wenn dies als Operator gewählt wird, muss eine '0' bei Sortierungswert eingetragen werden.
Möchten Sie beide Filter kombinieren, muss die Checkbox angewählt werden.
Zeilenauswahl:
Die Zeilenauswahl kann unabhängig von den anderen beiden Filtern verwendet werden. Sie können hier die Zeilen auswählen, innerhalb welcher die Filter angewendet werden sollen. Die gewünschten Zeilen müssen dabei im Format 'x-y' angegeben werden.
Die ursprüngliche Datei wird durch die Filter nicht verändert, Sie können diese jederzeit rückgängig machen. Möchten Sie die Änderungen speichern, können Sie einen Namen für das erstellte Subset eingeben, und dieses mit speichern. Nach einem Refresh ist die Datei dann im Dateiarchiv verfügbar.
Die Datei, die im Dateiarchiv ausgewählt ist, wird mit einem Klick auf mit auf die Detailseite genommen, wo Sie dann genauere Diagramme erstellen können!

Detailseite

Auch auf der Detailseite sehen Sie die ausgewählte Datei.
Wahl der Diagrammart:
Bei diesem Dropdown-Menü haben Sie die Möglichkeit zwischen einem Balkendiagramm, Liniendiagramm, Tortendiagramm, einer Wordcloud oder einer Wortartenanlyse zu wählen. Bei Wahl eines Balken-, Linien- oder Tortendiagramms müssen im nächsten Schritt noch die Spalten ausgewählt werden, die für das Diagram verwendet werden sollen. Bei Balken- und Liniendriagramm sind die ausgewählten Spalten dann die x- bzw. Achse. Beim Tortendiagramm wird angegeben wie viel % die Werte der y-Achse-Variable der x-Achse einnehmen. (Beispiel: Eine Tabelle mit Teams und Punkten, Team A: 25Punkte, Team B: 20 Punkte, Team C: 10 Punkte Team ist yAchse, und Punkte is xAchse -> Kuchenstück von A hat 46%, Kuchenstück von B hat 37%, Kuchenstück von C hat 17%) Eine Wordcloud zeigt Ihnen, welche Wörter am meisten in Ihrer Datei vorkommen. Eine Wortartenanlyse analysiert die Wortarten in der Datei.
Wahl der Spalten für die Achsen:
Nun haben Sie die Möglichkeit, ihre Spalten zu wählen, zu denen das Diagramm erstellt werden soll. In dem Dropdown-Menü für die y-Achse werden Sie nur die Spaltennamen finden, die ausschließlich numerische Werte enthalten, da sonst die Erstellung eines Diagramms nicht möglich ist.
Durch Klicken auf wird das Diagramm erstellt, und Ihnen auf der Detailseite angezeigt. Sie können dies so oft sie wollen wiederholen, oder über die Navigationsbar zurück auf die Übersichtsseite gehen, um eine andere Datei auszuwählen!
